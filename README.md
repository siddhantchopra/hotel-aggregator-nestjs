# Hotel Aggregator Application
**It is built on Nest.js framework with CRUD operations API related to create, retrieve, update and delete**

### Implementation of Hotels and Users Auth
## Application Integration

- Monorepo Structure
- Database: MongoDB
- Container: Docker
- Documentation: Swagger

### How to run the Application in your local machine

**Pre-requisite is docker / docker-desktop**

#### Commands to run the Application at your ease

- In your Terminal, paste this command - *docker-compose up --build -V*
- This will serve up your container and install all the dependencies, Serve up Nestjs App and mongoDB
- in your browser, visit this URL - *http://localhost:3000/api/hotel* (for hotel-services)
- in your browser, visit this URL - *http://localhost:3001/api/users* (for user-services)
- The above URLs will serve up your documentation via swagger and you can play around with the APIs and the data stored will be persisted by the container
